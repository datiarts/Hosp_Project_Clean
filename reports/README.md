# Reports Directory

This directory will contain exported reports from the AMP system.

## Report Types

- **Weekly Reports** - تقارير أسبوعية
- **Monthly Reports** - تقارير شهرية
- **Executive Summaries** - ملخصات تنفيذية
- **KPI Reports** - تقارير مؤشرات الأداء

## File Formats

- Excel (.xlsx)
- PDF (when implemented)

## Note

Reports are generated automatically by the system based on data from all modules.
